Check-in Manager

The Check-in manager helps the provider in managing the working hours and ensuring the seamless workflow from the point of the check-in to the completion of service.

Accept Online Check-in

  • Here, a provider gets full control over online client check-in.
  • Complete the “Profile & Summary” section with details. To do so, click on the “Profile” under Settings and enter the details. Once done, set Public Search to “ON”.
  • Afterward, a provider can enable “Online Check-in”.
  • Provider can choose two options:
    • Same day check-in : Check this button to accept check-ins for the same day.
    • Future check-in : Check this button to accept check-ins for a future date.
  • Walk-in check-ins are always enabled.

Calculation of waiting time

Waiting time simply refers to the time a client has to wait before he/she can avail the services offered at the premise.

  • Automatic (Machine Learning) : Using a machine learning algorithm, the system automatically calculates the turnaround time for each service by analyzing a provider’s service history.
  • Manual : Using this option, a provider can fix a wait time which will be applied to all the check-ins (for example, If a provider sets 10 minutes of manual time, by default all the clients will get 10 minutes of waiting time.) Manual time should be used when a provider can guarantee to serve all the clients in the given time. To do so, it is advisable that the service provider is equipped with enough resources. An additional option is available for providers choosing “Manual” wait time calculation. He/She can edit individual check-ins wait time from the check-in details in provider dashboard.
  • Token System(Do not show waiting time for clients) : If a service provider chooses this option, then all the clients will receive and see a token number instead of any waiting time.
  • No Token : If a service provider chooses this option, the clients won’t be able to see the waiting time as well as the token number. Here a client can see the number of persons ahead of him/her.

Send Notification

If a provider wants to receive notification in Jaldee Provider App, when a check-in is done or when a check-in is cancelled, he/she needs to turn on this toggle button.

Accept Future Check-In

  • This advanced feature of Jaldee.com helps a provider accept check-ins online for a service that will be availed in a future date.
  • If a provider enables this button, online check-in clients can avail the services for a future date
  • In case a provider disables“Accept Future Check-in”, clients can only check-in for the current day’s working hours.

LOCATIONS

  • The information related to the business location comes under this section.
  • This page prompts a provider to “Add new location”, in case he/she has not added any location during the initial profile set up.
  • “Add new location” allows the provider to add a new location for the enterprise.
    1. When the provider clicks on this button, a location window pops up where one can select a location on the map.
    2. Click “Yes I’m Done” after choosing a location in the map, then enter location name and rest of the fields will be filled automatically. If one cannot find the new location in the map, the provider will be required to manually enter the location details in the respective fields.
    3. In Working Hours the provider is required to add time schedule (Please refer Working Hours section below) and click on “Save” to add the new location.
    4. Click “Save” to submit all the information.
  • The location details entered during the time of the profile activation process is by default set as the base location of the business.
  • Multiple locations that are added in the profile is displayed here.
  • To set a particular location as the base location, click on the three dots menu in the right corner of the location name and click “Set as Base Location”.
  • The provider can make added location open/close to clients. To close a location for clients, click on the “Disable” button from the three dots menu in the right corner of the location name.
  • Click on the location name to view the details of a location including the location amenities and working hours.
  • The provider can also edit the location and location amenities of the enterprise.
  • Working hours (please refer Working Hours section below) of the location is displayed in the bottom of the page.
  • Multiple activities related to the Working Hours such as adding another working hours, editing the existing working hour, enabling/disabling the working hours can be done by clicking on the three dots menu.

SERVICES

This page lists all the services offered to the clients. While listing the services, please keep the name of services simple and comprehensible which will help the clients easily know more about the services offered by the clinic.

  • When the provider clicks on the “Services”, a page appears where one can view a service by default name “Facial”(If provider has edited the default service during the initial profile set up, the updated service details is seen here). Other details such as Estimated duration, End of Service notification fields also have values by default. The provider can edit this section by clicking on the pencil button.
  • Tax Applicable field can be set to “Yes”, only if tax settings is completed under the Payments section in Settings.
  • Service status is enabled by default. Only enabled services are available for clients to proceed with the check-in process.
  • The provider, at his/her convenience, can enable/disable the status of any service by clicking on the three dots menu.
  • All the services created are displayed here.
  • To view the complete details of a service, click on the service names. One can edit the details only if the status of service is set to enabled. In the detailed view, the provider can enable or disable the service’s status.
  • No price is set for any default service created. The provider can input the price in the specified field.
  • “Add Service” button enables the provider to add a new service to the profile. When a provider clicks this button, a service window pops up where one can input the service details
    • Service Name : Enter the name of the service.
    • Briefly describe service : Provide a brief explanation of the service that is being offered.
    • Estimated Duration (min) : Enter the minimum time (in minutes) required to complete the mentioned service.
    • Price : Enter the price of the mentioned service.
    • Enable Prepayment : A provider can enable the prepayment option. To accept payment in advance before checking in for the offered services, enable this option. Prepayment option can be enabled only when the payment settings are completed (Settings -> Payments).
    • Tax Applicable : If a provider has enabled the Tax application button, he/she can collect a certain percentage of tax from the clients (the percentage of tax can be set under Tax settings). This option can be enabled only if the Tax settings are already set.
    • End of Service Notification : When a provider enables the notification button, two options, viz, Push messages and Email appear. A provider can select any of these media to pass a message to the clients informing that the service has been completed.
    • Gallery : Multiple images of the service are added here.
  • Click on “Save” after adding all the information. A success window appears informing that the service has been added successfully and will ask to add this service to the relevant working hours.
  • A newly added services will be displayed to the clients, only if the service has been added to any existing/new working hours. This can also be edited by clicking on the pencil button.

WORKING HOURS

Working Hours simply means the active time of the business. One can add multiple Working Hours according to the business schedule. To understand this with an example, suppose a business works from Monday to Friday (9 am to 5 pm) whereas, on Weekends (i.e. Saturday and Sunday), it operates only for 4 hours (9 am to 1 pm). In such cases, one can create two working hours, one for weekdays and the other one for the Weekends.

  • This page prompts to “Add working hours”, in case the provider has not added any working hour during the initial profile set up.
  • “ADD WORKING HOURS” button enables the provider to add new working hours. When a provider clicks this button a window appears where one can input the details of the schedule.
    • Working Hours : Enter a title for the working hours here.
    • Location : Choose a location from the drop-down menu.
    • Service : All added services offered by the provider are displayed here. Check the services offered in this working hour from the list of services displayed here.
    • Schedule : Select the days according to the schedule. A provider has the option to choose “Select all days”, in case the business is open 24*7.
    • Start Time/End Time : Enter the start time and the end time for a particular working hour.
    • Maximum patients served : Enter the maximum number of clients the business can serve in particular schedule. No more check-ins will be accepted, once the limit is reached. If the limit is not set, then endless check-ins will be accepted between the working hours.
    • Clients served at a time : Here enter the number of clients that can be served at the same time.
    • Online check-ins : Enable this toggle button to make this particular working hours available for clients to check-in online for the same day.
    • Future check-ins : Enable this toggle button to make this particular working hours available for clients to check-in online for a future date.
  • All added working hours are displayed here.
  • The status of working hours can be enabled/disabled. The enabled working hours are open to clients for the check-in process.
  • Providers can enable or disable same day online check-in or future check-in anytime from here.
  • To edit or to enable/disable any particular working hour, click on the three dots menu.
  • Also “Show Checkins Count” is available in the three dots menu, to see the number of check-ins for Today, Future and Tomorrow.
  • When the service provider clicks on the enabled working hours, it displays a brief outlook of the time schedule.
  • To edit the timing of this particular working hours, click on the pencil button given in the right corner.
  • A working hour window appears where the fields show current data. Edit the sections and click on the “Save” button to retain the modified data.

My Availability Today(+)

In Jaldee, if provider is available to provide services to clients other than the available working hours, he/she can create an instant working hour for a time period other than the saved working hours.

Step by step instructions to add an instant working hour in Jaldee:

  • Click on (+) symbol, adjacent to the label “My Availability Today”.
  • Edit the time period you are available to offer services.
  • Check the services listed, you wish to offer during this instant working hour. If all services are offered, check the All Services checkbox.
  • Option to edit maximum clients served and clients served at a time, is available.
  • Finally, click “MAKE ME AVAILABLE” button to create an instant queue.
  • Now client check-ins can be accepted for this instant queue.

DEPARTMENTS

A corporate structure consists of various departments which performs specific functions while constantly collaborating with each other to achieve specific goals. In Jaldee, provider can add in departments and can allow client check-ins department wise.

How to add a department?

  • Initially, click “Add Department” button.
  • Enter the department name, department id and give a brief description.
  • Click “Save” to add the department.
  • Most importantly, add available services to each department.
  • The added departments are listed here. provider can enable or disable added departments from three dots menu.
  • To edit department details, click on the department name.
  • To allow clients to check-in department wise enable the toggle button “Select services by department” from check-in manager.
