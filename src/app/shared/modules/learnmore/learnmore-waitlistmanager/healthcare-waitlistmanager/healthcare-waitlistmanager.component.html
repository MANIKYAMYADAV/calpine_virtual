Waitlist Manager - Healthcare

Waitlist manager helps the provider to manage the service time windows and ensure seamless workflow from the point of the check-in to the completion of service.

ACCEPT ONLINE CHECK-IN

  • This gives you full control over online patient check-in.
  • First you have to complete “Profile & Summary” section under Profile & Search(Settings) and set Public search to “ON”.
  • And then you can enable “Accept Online Check-In”.
  • If this button is enabled, online check-in is open for patients and if this button is disabled, patients can only view your profile at their dashboard, but cannot check-in online for any services offered.
How is waiting time calculated ?

Waiting time refers to the time patient waits to avail his/her service offered at your enterprise.

  • Automatic (Machine Learning) : Our system uses machine learning algorithm which automatically calculates the turnaround time for each service by analysing provider’s service history.
  • Fixed : By selecting this option you can choose a fixed service time which will be applied for all check-ins.(eg. If you set 10 mins of fixed time, by default all the patients will get 10 mins of waiting time). Fixed time can be used only when you guarantee to serve all patients in the given fixed time. You should have enough resources to do the same.
  • Do not show waiting time for patients(Token Enabled) : If you enable this option, no wait time will be shown to your patients, but token number will be shown.
  • Do not show waiting time for patients(Token Disabled) : If you enable this option, neither wait time nor token number will be shown to your patients. But persons ahead is shown to get an idea of your turn for actual appointment.

Accept Future Check-In

  • This advanced feature helps you to accept check-ins online for a service in future.
  • If you enable this button, online check-in patients can avail services for a future date.
  • If you disable “Accept Future Check-In”, patients can only check-in for present day’s service time windows.

LOCATIONS

  • Your business location data comes under this section.
  • This page will prompt you to “Add new location”, if you have not added any location during the initial profile set up.
  • “Add new location” allows you to add a new location for your enterprise.
    1. When you click this button a location window pops up where you can select the location in map.
    2. Click “Yes I’ am Done” when you pick your location from map, then enter location name and the rest of the fields are filled automatically. If new location is not available in map all the fields should be entered manually.
    3. Then comes the Business Schedule section where you are asked to add time schedule(Please refer Service Time window section below) and click ‘Save schedule’ to add the new location details.
    4. Click “Save” to submit all the information.
  • Location details you have saved at the time of profile activation process is set by default as the base location of your business.
  • Multiple locations added is displayed here.
  • If you want to set a particular location as base location, click the three dots menu in the right corner of the location name and click “Set as Base Location”.
  • Also if you don't want to make any added location open to patients you can click “Disable” button from the three dots menu in the right corner of the location name.
  • Here, “Add new location” allows you to add a different location for your enterprise.
    1. When you click this button a location window pops up where you can select the location in map.
    2. Click “Yes I’ am Done”when you pick your location from map, then enter location name and the rest of the fields are filled automatically. If new location is not available in map all the fields should be entered manually.
    3. Then comes the Business Schedule section where you are asked to add time schedule(working hours) and click ‘Save schedule’ to add the new location details.
    4. Click “Save” to submit all the information.
  • When you click the location name, complete details of your location, location amenities and working hours are displayed.
  • Here you can edit the location and location amenities of your enterprise.
  • Working hours(Please refer Service Time window section below) of selected location is displayed in the bottom of the page.
  • You can add, edit and “Enable/Disable” service time window by clicking the three dots menu.

SERVICES

This page lists all the services you offer to your patients. Kindly note that service name should be simple and understandable which will help your patient to know more about the services you offer.

  • When you click “Services”,a page appears where you can view a service by default named “Consultation”. Estimated duration, Price, Tax Applicable, End of service notification fields have values by default. This can be edited clicking pencil button.
  • Tax Applicable field can be set to “Yes”, only if tax settings is completed under the Payments section in Settings.
  • Service status is also enabled by default. Only enabled services are available for patients to proceed with check-in.
  • You can switch the status button of any service either to Enabled or Disabled by clicking the three dots menu.
  • All the services created are displayed here.
  • When you click any service name, complete details of the service is displayed and can edit only if the status of service is set enabled. Also can disable or enable the service’s status from the detailed view.
  • For any default service created, no default price is set. You can input the price in the specified field.
  • “ADD NEW SERVICE” button enables you to add a new service to your profile. When you click this button a service window pops up where you can input the service details.
    • 1 Service Name Enter the name of the service.
    2 Description Give a description about the service you are offering to patients.
    3 Estimated Duration Enter the minimum time in minutes to complete this particular service.
    4 Price Input the price of this service.
    5 Enable Prepayment You can enable prepayment to accept payment in advance for the service offered. If you enable prepayment, patients are asked to pay in advance before checking-in for this particular service. Prepayment can be enabled only if you have completed the payment settings(Settings-->Payments).
    6 Tax Applicable If you enable the Tax applicable button you can collect a certain percentage of tax from the patients (The percentage of tax can be set in tax settings). This can be enabled only if Tax settings is already set.
    7 End of Service Notification When you enable the notification button, two options appear, Push Message and Email. Select any of these medium through which a message is passed to patients informing their service has been done.
    8 Gallery Multiple images of the service can be added here.
  • When you have all the information click “Save”, a success window appears informing the service has been added successfully and will ask to add this service to the relevant service time window by editing the service time window.
  • A newly added service will be displayed to patients, only if the service has been added to any existing/new service time window and also service can be edited clicking the pencil button.

SERVICE TIME WINDOW

Service Time Window simply means the working hours of your enterprise. You can add multiple service time windows according to your business schedule. Let’s say from Monday to Friday you have 9am to 5pm and for Saturday and Sunday you have 9am to 1pm. In this case you can create two service time windows one for Monday to Friday and another for Saturday and Sunday.

  • This page will prompt you to “Add new service time window”, if you have not added any service time window during the initial profile set up.
  • “ADD NEW SERVICE TIME WINDOW” , button enables you to add a new service time window.When you click this button a service time window pops up where you can input the service time window details.
    1 Service Time Window Name Enter a name for the service time window.
    2 Location Choose location from the drop down box.
    3 Service All added services are displayed here. Check the services offered in this service time window from list of services displayed here.
    4 Schedule Select the days according to your schedule . You can check “Select all days” check box if your business is open through the week.
    5 Start Time/End Time Enter the start time and end time for this service time window.
    6 Maximum Capacity Enter the maximum number of patients you can serve in this particular service time window. When the limit exceeds check-ins will not be accepted. And if limit is not set endless check-ins are accepted between the working hours.
    7 Number of patients served at a time Enter the number of patients served at the same time.
  • All added service time windows are displayed here.
  • Service time window’s status can be switched either to “Enabled” or “Disabled”. Only enabled service time windows are open to patients for checking-in.
  • You can “Enable/Disable” particular service time window by clicking the three dots menu. Also can edit the service time window from here.
  • When you click the enabled service time window brief outlook of this time schedule is displayed.
  • You can edit the timing of this particular service time window when you click the pencil button in the right corner.
  • When you click to edit the schedule, service time window appears where the input fields are filled with current data which can be edited. When you have all the right information click “Save” button.
