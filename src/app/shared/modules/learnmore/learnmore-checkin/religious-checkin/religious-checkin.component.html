Check In

  • Here you can create check-in for a devotee for a service.
  • You have to search the devotee with his mobile number, first name/last name search for this devotee whether he/she is already a registered customer of Jaldee.
  • If the devotee does not exists you can input the devotee details and get registered with Jaldee.
  • In check-in window first select the service you wish to avail from the drop down menu. If the provider has only one default service there will be no drop down menu.
  • Check-in window has a feature to add or change a family member whom you wish to do check-in for. Click “Add/Change Member” to add new member with first name and last name and also can change member from the list of members added already.
  • If you have something to write to the provider you can click “If you have a note, Click here(+)” and can write the message in the textbox.
  • With this information you are ready to proceed with online check-in by clicking “CONFIRM CHECK-IN”.
  • Push message will be sent to customer’s mobile and an email notification to the registered email id informing the online check-in.
  • Alike Check-in, Future Check-in can be done in the same way.The only difference is that you need to select the date of check-in here.
