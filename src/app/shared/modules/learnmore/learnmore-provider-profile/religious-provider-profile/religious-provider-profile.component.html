PRIEST PROFILE

The priest profile is a collection of settings and information associated with a business owner. The system creates a priest profile at the time of registration. The profile contains information such as name, mobile, email, etc. Jaldee priest profile has the following options:

  • My Account : Here lies the personal data of the service priest. Option to edit the data is also available.
  • Change Password : Here the priest can change the password keeping in line with the following guidelines:
    • Minimum 8 characters
    • Maximum 32 characters
    • Must have at least one uppercase letter and one digit.
  • Change Mobile : Enter a new mobile number and click on Submit. To verify the mobile number, enter the OTP sent to the number.
  • Switch to Consumer : When a priest profile is created in Jaldee.com, a corresponding devotee profile also gets created automatically for the same priest. From here, the priest can easily switch to the devotees dashboard.
  • Logout : One can log out of the account here.
