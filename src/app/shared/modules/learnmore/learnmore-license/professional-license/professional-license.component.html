Upgrade License

Here a provider can upgrade the license anytime.

  • When a provider clicks on the “Upgrade License” button, a window appears where a drop-down menu is listed with the available license packages.
  • To upgrade to a license package, select a package from the drop-down menu.
  • The license package has a monthly subscription, i.e. monthly payment has to be done.
  • A provider can also upgrade the license package using the below links from dashboard:
    • Click here to Upgrade License : If license package is Platinum Free 60 days trial.
    • Upgrade License Now : If license package is Basic or higher.
  • Here, a provider can view the owned license package, due amount as well as the next payment date.
  • An invoice is generated on a monthly basis.
  • For example, if a provider has created a profile on 10th January then all invoices will be generated on a monthly basis i.e. on 10th of each month. In case, the provider buys add-on and Jaldee-keywords on a different date (suppose on 20th January), the first bill of the add-on and Jaldee-keywords will be calculated from 20 January - 10 February. After that, the same “10↔10” bill cycle will continue from next month. This billing cycle will include License and Addon amount in a single invoice until the license or Add-on gets upgraded/downgraded.

    *To downgrade the existing license package, a provider needs to contact the Jaldee team, call us on 0487-2325650 or you can send us an email at support@jaldee.com.

  • An invoice is generated when an addon is included in a subscription using “ADD-ON(+)” option.
  • When a provider click on the “Pay” button (Silver and Higher grades), he/she can see the generated invoices. To proceed with payment from here, click on “Make Payment”. This redirects the page to PayU interface and thus proceed with the actual payment.
  • The provider will receive a confirmation email informing the payment status.
  • A provider can see all the payments completed so far in the “Payments History” section.
  • The invoice is generated every month on the same date of profile creation, whether or not the previous payment is made.
  • The license gets upgraded as soon as a provider clicks on license upgrade button, but the invoice gets generated from the next month and the amount is calculated based on the daily usage of the license package.
  • An email will be sent to the provider upon each invoice generation.

The three dots menu in the top right corner of the page has the following options:

1. License Change History

The license change history shows the list of the changes applied in the license package. Here a provider can see the starting date of the current license package, it’s expiry date (only if upgraded to higher grades), and the status of the license package either active or expired.

A provider can also downgrade the license via Jaldee support. The effects of this on the bill is reflected from the next bill cycle (next month) onwards.

2. License Usage

Here a provider can track the license usage. This section shows the following information:

  • Number of business profile views : This refers to the number of clients who have clicked for a detailed view of the service provider.
  • Transactions : This refers to the number of check-ins.
  • Profile gallery storage : This shows the total storage space given to the provider under the availed license package.
  • Jaldee-Keywords : This shows the total number of Jaldee-Keywords allowed for the availed package.

A provider also gets to know whether the usages are in compliance with the current license package. The data until yesterday is updated here.

3. Payment History

Here a provider can view the entire history of the payments done to Jaldee so far.

Add-on

Add-ons are used when a license package reaches its maximum limit and a provider wants to add more features. With the help of add-ons, a provider can avail additional features without upgrading the license. Add-ons are charged accordingly.


Following are the list of add-ons:

Storage per month

  • 250MB for Rs 100 per month
  • 1GB for rs 250 per month
Keywords
  • Pack of 5 keywords for Rs 250 per month
  • Pack of 20 keywords for Rs 1000 per month

*Note: Add-on deletion is possible only via Jaldee support. Once it is deleted, its effect on the bill can be seen from the next bill cycle onwards.

Jaldee-Keyword

Jaldee-Keyword is common keywords. A client generally uses such keywords to search for a service provider. Buying relevant keywords that a prospective client would be searching on Jaldee. For example, if yours is a business consultancy, you can buy Jaldee-Keywords like “Business Consultancy”, “Business Planning”, etc.
