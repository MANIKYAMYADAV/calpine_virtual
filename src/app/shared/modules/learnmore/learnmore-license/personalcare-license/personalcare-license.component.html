  • You are granted a license selected at the time of initial sign up window to use Jaldee for a period of time.
  • When you click License & Invoice from Settings, a window appears where current license package and related data is shown.
  • Jaldee under the license package “Gold_60Day_Trial” has all the attributes as that of “Gold” license package. This will automatically be expired in two months unless you upgrade the license packages to higher grades.
  • Alert messages will be received one week before the package expires.
  • Gold_60Day_Trial, is free of cost and can get to know the features of Jaldee.
  • All features you have been using in this period will expire and automatically the license package is upgraded to “Basic”.
  • And also Basic license package is free of cost with free limited features.

Upgrade License:

You can upgrade your license here anytime.

  • When you click the Upgrade License button a window appears where a drop down menu is listed with available license packages.
  • Select the license package, along with the pricing from the drop down which you wish to upgrade.
  • When you upgrade your license to any available packages, you are subscribed to the package on monthly basis.
  • You can also upgrade license package using the link “Upgrade Membership”, adjacent to the provider name in the top right corner of the page.
  • Your current license package is shown in the page. Due amount and Next Payment date is displayed here.
  • When you click “Pay” button(Silver and higher grades) you can see the invoices generated and also can proceed with payment from here clicking “Make Payment”. This will redirect you to PayU interface and you can proceed with actual payment.
  • A confirmation email will be sent about the payment status.
  • In the Payments History you can see the payments done so far.
  • An invoice is generated when:
    • license package is enrolled to higher grades.
    • an addon is included in a subscription using ADD-ON(+) option.
    • AdWords are added.
    • the payment is done every month on the same date of profile creation.
    • an email will be send to providers upon each invoice generation.

The three dots menu in the top right corner of the page has following three options.

License Change History

The license change history shows the list of the changes applied in the license package. Here you can see the starting date of using any license package, it’s expiry date(only if upgraded to higher grades), and the status of license package either active or expired.

License Usage

Here you can track the license usage. This section will show the informations about number of business profile views, transactions, profile gallery storage and AdWords. You will also get the information whether the usage are in compliance with the current license package. The data, till the day before, is updated here.

Payment History

Here you can view the complete history of the payments done to jaldee so far.

ADWORDS

Adwords are common keywords, a consumer might use to search for a particular provider. Buy relevant AdWords that a prospective customer would be searching on Jaldee. For example if your business is a Hair Salon you can buy keywords like “Hair”, “Haircut” etc.
