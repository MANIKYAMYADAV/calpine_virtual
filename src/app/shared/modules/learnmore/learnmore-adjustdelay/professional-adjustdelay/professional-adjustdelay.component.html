ADJUST DELAY

Delay can be caused by any reason. This interferes with the smooth functioning of the business. Adjust delay helps the provider to set a delay to the current or earliest working hour.

  1. Select a working hour from the dashboard to adjust the delay.
  2. Click on the “Adjust Delay” button. This opens a new window.
  3. Further, set the delayed time (in hours and minutes) in the Delay field.
  4. In case if the provider wants to inform the client about the delay, click “Send Message”. A default message will be sent to the clients informing that the wait time has been extended for some time along with the new estimated waiting time.
  5. The provider can also send a personalized message. Enter the message in the given text box.
  6. Click the “Save” button to confirm the adjustments.
