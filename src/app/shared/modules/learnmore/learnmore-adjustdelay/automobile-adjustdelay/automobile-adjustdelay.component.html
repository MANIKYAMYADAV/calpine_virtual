Adjust Delay

Delay can be caused for any reason which interferes the smooth functionality of the business. Adjust delay helps you to set a delay to present day’s service time window.

  1. Click Adjust Delay button, this should bring up a new window.
  2. Select a service time window from the drop down box where that day’s service time window are listed.
  3. Next in the Delay field set the time delay in hours and minutes.
  4. If you want to inform your patients about the delay caused click “Send Message” , a default message will be send to the patients or if you want to personalize the message, write the message in the textbox appeared.
  5. When you have all the information click “Save” button.
