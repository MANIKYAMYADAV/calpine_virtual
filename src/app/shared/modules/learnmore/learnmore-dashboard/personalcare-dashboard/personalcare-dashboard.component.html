Provider Dashboard-PersonalCare

You can monitor the current and future check-ins and also history of check-ins here. On the left side of the dashboard select the location and the service time windows available.

  1. Today () - Here the current day’s online, walk-in, and kiosk check-ins are listed. In the brackets appended to “Today” the total number of check-ins is displayed.
  2. Future() - Here the check-ins for future dates are listed. In the brackets appended to “Future” the total number of future check-ins are displayed.
  3. History - Here you can see all check-ins done till the day before. Check-in details can been seen clicking date of check-in or name of the client. Today’s check-in will be updated under the history tab only the next day. Actions also can be changed from here.

Location Selection

If your enterprise has multiple locations that has been added to your JALDEE profile, select one location to proceed with processing the Service time window. Please ensure the location you selected, before you check-in a walk-in client.

Check-ins

  • Select the location, if your enterprise has more than one location.
  • A detailed view of clients who have already checked in (online, walk-in or kiosk) is displayed.
  • Prefix to the name, if you see a “cloud” symbol, remember that particular client has checked in online.
  • When you click on token number or on the client name, you can view complete check-in details of the client. Check-in history, Communication History and Private Notes history are listed here. In addition to that a provision to cancel client check-in, communicate with client through Send Message and add private note are available.
  • The three dots menu in the right side of a particular check-in has the following options.
    1. Send Message : Write to your client from here. All these messages sent will be seen in the communication history anytime .
    2. Cancel check-in :
      • If you want to cancel a client’s check-in for any reason, you can click ”Cancel Check-In”.
      • clients are notified with push message/email, when their check-in is cancelled.
      • When you click ”Cancel Check-In”, a pop up window appears which shows few reasons listed with tick box . You can tick any valid reason from the list and click “Save”.
      • If you want to inform your client about the delay caused,tick the Send Message button and a text box appears with a default message respectively for each reason. You can edit the message and can type your own message.
    3. Add private note : If you want to keep a note about a particular client, you can write the note here. This record will be seen under Private notes, anytime this client checks-in again.
    4. Change Status to Check-in : This option appears only if the status is marked to “Arrived”. You can change the status back to “Check-in” with this option. This can be used under the situations:
      • when a client is cancelled for any reason and wanted to check-in again.
      • when a client check-in is accidently changed to Arrived status.
  • Each client’s Name and Service selected is displayed.

ACTIONS

There are four main actions that is performed here.

  1. Arrived : This button is used when a client arrives at the premise to avail his/her service. When you click this button, “Arrived” label is appended to the name of the client. For a walk-in or kiosk client this label is appended automatically as they are present at the premise to avail the service. Now the service can be started.
  2. Start : When the service is started click the Start button and the client entry is moved to “Started” tab. And also the other two buttons “Complete” and “Bill” opens.
  • Payment Status: Shows any payment status of the client, whether it is not paid, partially paid or fully paid.
  • Client Note: The notes (if any) added by the client while checking-in will be displayed here.

Started

  • Under this tab, the provider can view the list of clients who have already started their service.This client entry will no longer be seen in check-ins tab.
  • Name and Service is the same as in check-ins tab.
  • Mainly, two actions are performed here.
  1. Complete : When the service is done, you can click Complete button and the client entry is moved to “Completed” tab. If you have enabled ‘End of Service Notification’ button, either to Push Message or Email in the “Add New Service” page, message will be send to client informing the service has finished.
  2. Bill : When a service is started “Bill” button appears, which opens the bill of service checked-in for. You can add more services or items to the bill before payment, clicking “Add Service/Item”. The three dots menu near this button has options:
    • Apply Jaldee Coupon : Coupons provided by Jaldee.
    • Apply Coupon : Coupons added by provider.
    • Apply Order Discount : Discounts added by provider. Private Note and client note can be added from here before applying discount.
  • All these coupons are applied to Gross bill amount to get the total bill amount.
  • And discounts can either be applied to gross amount or each item.
  • Three dots near services in the bill has the option “Remove service” and items in the bill has options “Apply Discount” and “Remove item”.
  • In addition, Add private note, Send Message and Change Status to check-in options are also given.
    1. Add a provider note: If you want to keep a note about a particular client, you can write the note here.
    2. Send Message : Write to your client from here. All these messages sent will be seen in the communication history anytime.
    3. Change Status to check-in : This option appears only if the status is marked to “Arrived”. You can change the status back to “Check-in” with this option. This can be used under the situations:
      1. when a client is cancelled for any reason and wanted to check-in again.
      2. when a client check-in is accidently changed to Arrived status.
  • Payment Status: Shows any payment status of the client, whether it is not paid, partially paid or fully paid.
  • Client Note: The notes (if any) added by the client while Check-in will be displayed here.

Completed

  • Once the service is completed, that particular client entry is moved under this tab.
  • Accept Payments is an important choice here.
    1. Accept Payment : When you click this button, saved bill is displayed and can edit bill adding service or items. This bill has three options:
      • Settle Bill : When you click this button, the bill is settled and due amount is shown in the bill in client dashboard also.
      • Jaldee Pay : When you click this button, the bill is also generated at client dashboard, enabling them to pay online.
      • Pay by cash or others : When you click this button, payment can be done either by cash or by other means.

Cancelled

  • Here you can see the list of cancelled client check-ins.
  • The three dots menu has three options:
    • Add a provider note: If you want to keep a note about a particular client, you can write the note here.
    • Send Message : Write to your client from here. All these messages sent will be seen in the communication history anytime.
    • Change Status to check-in : The status can be changed back to check-in from here.
