PAYMENTS

In this section, a provider can select the payment mode with which he/she wishes to accept/settle payments.

  • To enable online payments using any payment system, a provider has to switch on the “Customers Self pay” mode. Initially, this is set to off mode.
  • If a provider has not entered an email address before, click on the link “Go To Email Settings” (Payment Settings) to add the email id. This can be set directly from My Account.

    • Payment Settings

  • Two modes of payments can be enabled. These are:
    • Jaldee Bank : Jaldee account means a customer can pay online for the services directly to the Jaldee Bank and in turn, the Jaldee team will reimburse the amount (charging a commission for each transaction) to the service provider. Jaldee bank is available for online payment. Upto Silver license package, only Jaldee bank is available for online payments. However, the providers with higher license packages (from Gold onwards) may or may not opt for the Jaldee bank for online payment.
    • My Own Bank Account : In this section, a provider is required to submit the account details of Paytm/CC/DC/Net Banking. In case the provider has not added an email address, click on the link “Go To Email Settings” (Payment Settings) to add the email address. Furthermore, a provider needs to click on any mode of payment (with which he/she wants to settle the bill). The mode of payment can be either “Paytm" or “CC/DC/Net Banking” or both.
      • Paytm : This is an effortless mode of online payment commonly used these days. If the provider has a Paytm account, he/she needs to provide the following details:
        • Mobile Number
        • Merchant Id
        • Merchant Key
      • CC/DC/Net Banking : To settle the bill using this mode of payment, a provider has to provide the following information:
        • Account Holder Name
        • Bank Account Number
        • Pan Card Number
        • Bank Name
        • IFSC Code
        • Name on Pan Card
        • Branch Name
        • Business Filing Status
        • Account Type
  • To verify the Paytm/CC/DC/Net Banking and to start accepting digital payments, please call the Jaldee team on 0487-2325650 or mail us to support@jaldee.com.
