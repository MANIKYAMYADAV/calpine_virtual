BILLING/POS - Professional Consulting

Billing/POS page is the one-stop-shop for adding items, discounts and coupons. Discounts and coupons play a key factor in driving clients towards a product/service and in boosting sales. Add item enables you to create and publish list of items you wish to sell at your premise. Jaldee also gives you the flexibility and ease of providing clients with discounts and coupons. Discount/Coupons can be redeemed to specific services/items and also to the total amount. You can set up two types of discounts/coupons in your Jaldee profile:

  1. Fixed Amount: This discount type will deduct the specified amount from either the total amount on the invoice or each specified item in the bill.
  2. Percentage: This discount type will deduct the specified percentage of the total amount from the invoice. "Percentage" type discounts/coupons can however be applied either to the total amount on the invoice or each specified item in the bill.
Add Item

To add a new item, click “Add Item”, a new window appears where you need to input information regarding the item. If the item is taxable, enable the “Taxable” toggle button(The percentage of tax can be set in tax settings). This can be enabled only if Tax settings is already set. Click “Submit” if you have all the information.

Add Discounts

To add a discount for the first time click “Add Discount”. Here you give the name of the discount, type of the discount(Percentage/Fixed) and provide a description to the discount. Discounts can be applied in following ways:

  • To every item/service.
  • To the total bill.

The Discount dashboard will show you all existing discounts that have been created on your profile. Here you can create, delete and edit a discount. For creating more discounts click (+) symbol.

Add Coupons

Here there are 2 main sections:

  1. Jaldee Coupons :
    • The coupons provided by Jaldee will be listed here. Each coupon has a coupon code, name, validity from date, to date, status and a three dots menu option to enable/disable the coupon.
    • The coupons which has no three dots menu are enabled by the Jaldee, for which provider has no right to enable or disable the coupon.
    • When you click the coupon code or the pointer, you can see the detailed view of that particular coupon.
    • In the detailed view, you can see the valid dates through which this coupon is available, discount value or percentage.

    The following are the rules for each coupons. This is marked either “❌” or “✅” which indicates whether it is applicable or not.

    1. Always Enabled : This coupons will always be enabled. Provider cannot disable or enable it.
    2. Default Enabled : This coupon will be enabled by default, but can change the status by the provider.
    3. Online Check in Required : This coupon will be available only for those clients who checks-in online.
    4. Self Payment Required : This coupon will be available only for those clients who pay online for the services.
    5. Combine with Other Coupons : This coupon can be used with other available coupons.
    6. First Check-in Only : This coupon will be available only for those clients who checks-in for the first time in Jaldee.
    7. First Check-in Per Provider Only : This coupon will be available only for those clients who checks-in for the first time for a particular provider.
    8. Consumer Use Limit Per Provider
    9. Minimum Bill Amount
    10. Consumer Use Limit
    11. Provider Use Limit
    • Terms and Condition and Provider description for this coupon is also seen in this page.
    • Corresponding client description will be seen to clients at their end.
    • Coupon status is seen in the bottom of the page, from here status can be enabled or disabled( only if permitted).
    • In addition a coupon stat section is given where the following is shown:
      1. Check ins Applied
      2. Total Amount Discounted
      3. Amount Reimbursed by Jaldee.com
  2. My Coupons

    These are coupons created by Providers. To add a coupon for the first time click “Add Coupon ”. Enter the name of the coupon, type of the coupon (Percentage/Fixed) and provide a description to the discount. Coupons can be applied in following ways:

    • To every item/service.
    • To the total bill.

    The Coupons dashboard will show you all existing coupons that have been created on your profile. Here you can create, delete and edit a coupon.For creating more coupons click (+) symbol.
