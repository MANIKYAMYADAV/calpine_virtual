DOWN PANEL OPTIONS

  • Here the doctor can quickly access the following options:
    • Alerts : All the Jaldee Broadcasts are shown here. These are the Jaldee provided notifications. Here a doctor can learn about new coupons or any other such information directly from Jaldee Super admin.
    • Audit Log : Here the doctor can view the activities done by him/her. The details appear with a date and time stamp. It also shows the actions completed by the doctor along with a little description.
    • Enable/Disable Check-in : This is a quick tab that helps the doctor to enable check-in, if disabled or to disable the check-in, if enabled.
    • License Details :Here the doctor can see the current License version he/she is using. This option directly leads to the license page, where the doctor can view the license history, its usage, payment history and upgrade the current License package.
