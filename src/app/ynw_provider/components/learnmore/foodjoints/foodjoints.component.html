Public Search
  1. A provider is required to click on the “TURN ON” button to enable the Public Search if he/she wants to be visible in the customer’s search results.
  2. If a provider has disabled the Public Search, customers cannot see the provider’s profile in their search results.
  3. Customers can search for a provider with any attributes that are listed under the Profile and Search section. These attributes can include the business name, Jaldee verified levels, Jaldee-keyword, Basic Information, Location, Location Amenities, Specialization, Languages Known, Additional Info.
  4. During the initial profile setup, a provider can proceed with setting the Profile Name & Summary, Location, Working Hours, Service and Public Search status. The provider can setup or update these informations here.
    • If the provider logs out of the website after setting the password, the account will not be active. To activate the account, the provider has to completely fill basic details and turn on the Public Search. At this point, a default service gets created which can be edited in Services under the Check-In Manager.
    • If the provider logs out of the website after setting up the location, a default service time window gets created. The provider can edit this from here.

    CHECK-IN MANAGER

      • Same day check-in : Check this button to accept check-ins for the same day.
      • Future check-in : Check this button to accept check-ins for a future date.
    • Walk-in check-ins are always enabled.
    Calculation of waiting time
    Waiting time simply refers to the time a customer has to wait before he/she can avail the services offered at the premise.
    • Automatic (Machine Learning) : Using a machine learning algorithm, the system automatically calculates the turnaround time for each service by analyzing a provider’s service history.
    • Manual : Using this option, a provider can fix a wait time which will be applied to all the check-ins (for example, If a provider sets 10 minutes of manual time, by default all the customers will get 10 minutes of waiting time.) Manual time should be used when a provider can guarantee to serve all the customers in the given time. To do so, it is advisable that the service provider is equipped with enough resources. An additional option is available for providers choosing “Manual” wait time calculation. He/She can edit individual check-ins wait time from the check-in details in provider dashboard.
    • Token System(Do not show waiting time for customers) : If a service provider chooses this option, then all the customers will receive and see a token number instead of any waiting time.
    • No Token : If a service provider chooses this option, the customers won’t be able to see the waiting time as well as the token number. Here a customer can see the number of persons ahead of him/her.

    Working Hours
    Working Hours simply means the active time of the business. One can add multiple Working Hours according to the business schedule. To understand this with an example, suppose a business works from Monday to Friday (9 am to 5 pm) whereas, on Weekends (i.e. Saturday and Sunday), it operates only for 4 hours (9 am to 1 pm). In such cases, one can create two working hours, one for weekdays and the other one for the Weekends.
    • This page prompts to “Add working hours”, in case the provider has not added any working hour during the initial profile set up.
    • “ADD WORKING HOURS” button enables the provider to add new working hours. When a provider clicks this button a window appears where one can input the details of the schedule.
      • Working Hours : Enter a title for the working hours here.
      • Location : Choose a location from the drop-down menu.
      • Service : All added services offered by the provider are displayed here. Check the services offered in this working hour from the list of services displayed here.
      • Schedule : Select the days according to the schedule. A provider has the option to choose “Select all days”, in case the business is open 7 days a week.
      • Start Time/End Time : Enter the start time and the end time for a particular working hour.
      • Maximum patients served : Enter the maximum number of customers the business can serve in particular schedule. No more check-ins will be accepted, once the limit is reached. If the limit is not set, then endless check-ins will be accepted between the working hours.
      • Customers served at a time : Here enter the number of customers that can be served at the same time.
      • Online check-ins : Enable this toggle button to make this particular working hours available for customers to check-in online for the same day.
      • Future check-ins : Enable this toggle button to make this particular working hours available for customers to check-in online for a future date.
    • All added working hours are displayed here.
    • The status of working hours can be enabled/disabled. The enabled working hours are open to customers for the check-in process.
    • Providers can enable or disable same day online check-in or future check-in anytime from here.
    • To edit or to enable/disable any particular working hour, click on the three dots menu.
    • Also “Show Checkins Count” is available in the three dots menu, to see the number of check-ins for Today, Future and Tomorrow.
    • When the service provider clicks on the enabled working hours, it displays a brief outlook of the time schedule.
    • To edit the timing of this particular working hours, click on the pencil button given in the right corner.
    • A working hour window appears where the fields show current data. Edit the sections and click on the “Save” button to retain the modified data.

      9. Minimum Bill Amount : This refers to the minimum bill amount to which the coupon can be applied.
      10. Consumer Use Limit : This refers to the number of times a customer can apply the coupon.
      11. Provider Use Limit : This refers to the number of times a provider can apply the coupon.
      • Here a provider can view the terms and conditions as well as the provider description for the coupon.
      • Corresponding customer description is displayed to him/her at their dashboard.
      • Coupon status is visible in the bottom of the page. The provider has the option to enable/disable the coupon here (only if permitted).
      • In addition, a coupon stat section is given where the following is shown:
        1. Check ins Applied : This shows the number of times check-ins applied for the coupon.
        2. Total Amount Discounted : This shows the total amount waived to the customers by the provider using the coupon.
        3. Amount Reimbursed by Jaldee.com : This shows the amount given to the provider by Jaldee.
         My Coupons
    These coupons are created by providers. To add a coupon for the first time click “Add Coupon”. Here enter the name of the coupon, type of the coupon (Percentage/Fixed) and provide a brief description to the discount. A coupon can only be applied to the total bill. A provider can set up two types of Coupons:
    1. Fixed Amount: A specified amount will be deducted from the total bill.
    2. Percentage: A specific percentage of the amount from the total bill will be deducted in this case. “Percentage” type coupons can be applied to the total bill.
    The Coupons dashboard shows all the existing coupons that have been created on the profile. Here a provider can create, delete and edit a coupon. To create more coupons, click on the “(+)” symbol.
    Tax Settings
    Before setting up the taxes, ensure that one is familiar with the tax requirements of one’s business. The following instructions will help the provider through the tax setup of the business services in Jaldee:
    • When the provider clicks on the tax settings under the Billing/PoS section, a window appears. The provider is required to fill the textbox (Enter the GST number of the business and the tax percentage for the respective sector here).
    • At first, enter the GST number of the business.
    • Next, enter the Tax Percentage.
    • After the complete insertion of the details, click “Save”.
    • This tax will be applied to the services offered and the items that are being sold at the time of billing, only if the “Tax Applicable” toggle button is enabled in the respective section (i.e. Settings -> Check-In Manager ->Services).
    • Jaldee has no role in paying taxes paid by the customer to the service provider. All types of taxes are collected directly by the service provider, so it is the provider’s responsibility to remit the taxes to the Government.
    PROVIDER PROFILE
    The provider profile is a collection of settings and information associated with a business owner. The system creates a provider profile at the time of registration. The profile contains information such as name, mobile, email, etc. Jaldee provider profile has the following options:
    • My Account : Here lies the personal data of the service provider. Option to edit the data is also available.
    • Change Password : Here the provider can change the password keeping in line with the following guidelines:
      • Minimum 8 characters
      • Maximum 32 characters
      • Must have at least one uppercase letter and one digit.
    • Change Mobile : Enter a new mobile number and click on Submit. To verify the mobile number, enter the OTP sent to the number.
    • Switch to Consumer : When a provider profile is created in Jaldee.com, a corresponding customer profile also gets created automatically for the same provider. From here, the provider can easily switch to the customers dashboard.
    • Logout : One can log out of the account here.
    DOWN PANEL OPTIONS
    • Here the provider can quickly access the following options:
      • Alerts : All the Jaldee Broadcasts are shown here. These are the Jaldee provided notifications. Here a provider can learn about new coupons or any other such information directly from Jaldee Super admin.
      • Audit Log : Here the provider can view the activities done by him/her. The details appear with a date and time stamp. It also shows the actions completed by the provider along with a little description.
      • Enable/Disable Check-in : This is a quick tab that helps the provider to enable check-in, if disabled or to disable the check-in, if enabled.
      • License Details : Here the provider can see the current License version he/she is using. This option directly leads to the license page, where the provider can view the license history, its usage, payment history and upgrade the current License package.
